# DigiTools – Premium Digital Tools Platform

A modern single-page digital product marketplace built with React + Vite + Tailwind CSS.

## Features
- Browse premium digital tools
- Add to cart with toast notifications
- Cart management (add, remove, total calculation)
- Responsive design (mobile + desktop)
- Beautiful UI with gradient accents

## Tech Stack
- React + Vite
- Tailwind CSS
- Lucide React (icons)
- React Toastify

## How to Run
```bash
npm install
npm run dev
